## 简介

一套企业级前后端分离系统解决方案。

服务端基于：[AspNetZero](https://github.com/aspnetzero)

客户端基于：[Ant Design](https://github.com/ant-design/ant-design)

UI样式基于：[antd-admin](https://github.com/zuiidea/antd-admin)

## 安装

```bash
npm install
```

## 运行

```bash
npm start
```
